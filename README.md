# Netflix-data-analytics
This project presents an interactive Netflix Data Analysis Dashboard built to explore and visualize content trends, user preferences, and platform insights. The dashboard provides a comprehensive view of Netflix's movies and TV shows using data-driven storytelling techniques
Project Overview

The goal of this project is to analyze Netflix's content library and uncover meaningful insights such as content distribution, genre trends, release patterns, and regional availability. The dashboard is designed to support data-driven decision-making and showcase data visualization skills.

📌 Key Features
📅 Analysis of content release trends over the years
🎭 Genre-wise distribution of Movies and TV Shows
🌍 Country-wise content availability
⏱️ Duration-based categorization (Movies vs TV Shows)
⭐ Identification of popular categories and patterns
📈 Interactive filters for dynamic exploration
🛠️ Tools & Technologies Used
Power BI – Data visualization and dashboard creation
SQL – Data extraction and transformation
Excel / Power Query – Data cleaning and preprocessing
📊 Insights Derived
Growth of Netflix content over time
Dominance of specific genres and countries
Comparison between Movies and TV Shows
Trends in content duration and ratings
📁 Dataset

The dataset used in this project contains details such as:

Title
Type (Movie/TV Show)
Genre
Release Year
Duration
Country
Rating
🎯 Objectives
To demonstrate data analysis and visualization skills
To build an interactive and user-friendly dashboard
To extract actionable insights from real-world data
